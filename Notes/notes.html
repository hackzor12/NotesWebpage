Advanced

  • Everything on a webpage is divided into a stucture of boxes.
  • HTML contains the stucture and the content of a webpage but not the style
  • The style of a webpage is contained in a CSS file and it interacts with the stucture to easily apply different styles to different boxes/divs/spans on a webpage
  • CSS allows us to style webpages easily and avoid repitition and it is important not to write the CSS inline to help avoid repitition.
  • The box model of every element on a webpage content > padding > margin > border
  • Box-sizing allows the box to stay the same by adjusting the margin keeping the box at a constant size
  • HTML and CSS should comply with standards for all common browsers
